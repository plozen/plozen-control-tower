# Styleguide

Visual proof pages for `DESIGN.md`. 이 디렉터리는 장식용 갤러리가 아니라 디자인 산출물 검수표입니다.

- `tokens.html`: color, typography, spacing, and viewport criteria.
- `sitemap.html`: design-system route map and output flow.
- `components.html`: component state matrix and compact specimens.
- `patterns.html`: app, tablet, web, admin, portfolio surface matrix.
- `states.html`: loading, empty, error, disabled, permission, offline, validation, unsaved, success states.

The styleguide is not a production app. It is a review surface for design decisions before implementation or portfolio export.
